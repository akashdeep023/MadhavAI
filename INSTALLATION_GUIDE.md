# MadhavAI App Installation Guide

## Download the App

### Option 1: Download from GitHub Releases (Recommended)

1. Visit the [Releases page](https://github.com/akashdeep023/MadhavAI/releases)
2. Find the latest release version
3. Download the APK file for your device:
   - **app-universal-release.apk** - Works on all Android devices (recommended for most users)
   - **app-arm64-v8a-release.apk** - For modern 64-bit ARM devices
   - **app-armeabi-v7a-release.apk** - For older 32-bit ARM devices
   - **app-x86_64-release.apk** - For Intel/AMD 64-bit devices
   - **app-x86-release.apk** - For Intel/AMD 32-bit devices

> **Tip:** If you're unsure which version to download, choose **app-universal-release.apk**

### Option 2: Build from Source

If you want to build the app yourself, follow the instructions in [QUICK_START.md](QUICK_START.md)

## Installation Steps

### Step 1: Enable Unknown Sources

Before installing the APK, you need to allow installation from unknown sources:

1. Open **Settings** on your Android device
2. Go to **Security** or **Privacy**
3. Find **Install unknown apps** or **Unknown sources**
4. Select your browser or file manager
5. Toggle **Allow from this source** to ON

> **Note:** The exact steps may vary depending on your Android version and device manufacturer

### Step 2: Install the APK

1. Locate the downloaded APK file in your **Downloads** folder
2. Tap on the APK file
3. Review the permissions requested by the app
4. Tap **Install**
5. Wait for the installation to complete
6. Tap **Open** to launch the app

### Step 3: First Launch

When you first open MadhavAI:

1. The app will request necessary permissions (camera, storage, location, etc.)
2. Grant the required permissions for full functionality
3. You'll see the login/registration screen
4. Create a new account or log in with existing credentials

## Troubleshooting

### Installation Blocked

If you see "Installation blocked" message:
- Make sure you've enabled installation from unknown sources (see Step 1)
- Check if Play Protect is blocking the installation:
  1. Open **Play Store**
  2. Tap your profile icon → **Play Protect**
  3. Tap the gear icon → Disable **Scan apps with Play Protect** temporarily
  4. Install the app, then re-enable Play Protect

### App Not Installing

If the installation fails:
- Make sure you have enough storage space (at least 100 MB free)
- Delete the old APK if you downloaded it before
- Restart your device and try again
- Make sure you downloaded the correct APK for your device architecture

### App Crashes on Launch

If the app crashes when you open it:
- Clear the app cache: **Settings → Apps → MadhavAI → Storage → Clear Cache**
- Uninstall and reinstall the app
- Make sure your Android version is 5.0 (Lollipop) or higher
- Check if you have a stable internet connection for first-time setup

### Permissions Issues

If the app is not working properly:
- Go to **Settings → Apps → MadhavAI → Permissions**
- Make sure all required permissions are granted:
  - Camera (for scanning and image capture)
  - Storage (for saving data offline)
  - Location (for weather and location-based features)
  - Microphone (for voice commands)

## Updating the App

When a new version is released:

1. Download the latest APK from the [Releases page](https://github.com/akashdeep023/MadhavAI/releases)
2. Install the new APK over the existing app
3. Your data and settings will be preserved

> **Note:** You don't need to uninstall the old version first
